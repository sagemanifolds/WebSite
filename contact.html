Help

Ask questions to ask.sagemath.org; please tag your question as sagemanifolds or manifolds

Technical questions regarding e.g. the installation of SageMath should be asked to sage-support.

Mailing list

News, bugs, ideas, general discussion: subscribe to sagemanifolds.list.

Contact info

Send a message to the project coordinator: Eric Gourgoulhon. See also the full list of authors.

Visit the contribute page if you would like to take part in the project.
